Dijkstra Algorithm
Tools & keywords JavaScript, Html&Css, Graph and Network Theory, Dijkstra Algorithm Simulation , Relaxation process for Dijkstra algorithm.
Details In this application I have created fonctions which handel the creation of Vertices and edges between them, And after creating the graph the user puts The start vertex, where here the application will generate a table for the shortest paths from the start vertex to all the other vertices, after that the user could input the vertex which he wants to know the shortest path to and click the button to see that all the edges from the start vertex to this specific vertex became in red color and with displaying more information concerning the path and the shortests distance.
